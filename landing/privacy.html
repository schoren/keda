Política de Privacidad

Última actualización: 16 de Enero, 2026

1. Introducción

RockingSoft SRL ("nosotros", "nuestro") opera la aplicación KEDA. Esta página le informa sobre nuestras políticas con respecto a la recopilación, uso y divulgación de datos personales.

2. Recopilación de Datos

No recopilamos datos personales para fines comerciales. Los datos que usted ingresa en la aplicación se utilizan exclusivamente para proporcionarle las funcionalidades de presupuesto y seguimiento de gastos.

3. Uso de Datos

Utilizamos sus datos para proveer y mantener el servicio, notificarle sobre cambios en nuestro servicio y permitirle participar en funciones interactivas cuando elija hacerlo.

4. Contacto

Si tiene preguntas sobre esta Política de Privacidad, por favor contáctenos.
