# Gestión de Cuentas

Las cuentas representan dónde vive tu dinero (por ejemplo, Banco, Efectivo, Tarjeta de Crédito).

## Ver y Crear Cuentas

Navega a la pestaña de **Cuentas** para ver tus saldos. Puedes añadir nuevas cuentas haciendo clic en el botón de **+** (añadir).
