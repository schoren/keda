# Instalar en iOS

## Progressive Web App (PWA)

Como KEDA aún no está disponible en el App Store, podés instalarlo como una Progressive Web App (PWA) en tu iPhone o iPad para tener una experiencia similar a una app nativa.

## Pasos para Instalar

1.  **Abrir Safari**: Navegá a [https://app.getkeda.app](https://app.getkeda.app) (o a tu instancia propia).
2.  **Tocar Compartir**: Tocá el botón de **Compartir** (el cuadrado con una flecha hacia arriba) en la parte inferior de la pantalla.
3.  **Agregar a Inicio**: Deslizá hacia abajo y tocá **Agregar a Inicio**.
4.  **Confirmar**: Tocá **Agregar** en la esquina superior derecha.

El ícono de KEDA aparecerá en tu pantalla de inicio y podrás abrirlo como cualquier otra aplicación.
