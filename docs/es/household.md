# Miembros del Hogar

Keda está diseñado para familias. Puedes compartir tus datos financieros con otros miembros de tu hogar.

## Gestionar Miembros

Navega a la pestaña de **Miembros** para ver quién tiene acceso a los datos del hogar.

![Pantalla de Miembros](/assets/members.png)

## Roles y Permisos

Todos los miembros invitados tienen privilegios de administrador. Pueden ver, añadir, editar y eliminar gastos, así como gestionar a otros miembros del hogar.

## Invitar Miembros

En la pestaña de **Miembros**, haz clic en **Invitar Miembro** para generar un enlace de invitación o enviar una invitación por correo electrónico.
