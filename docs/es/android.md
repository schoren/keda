# Instalar en Android

## Descarga Directa (APK)

Como KEDA aún no está disponible en el Google Play Store, podés descargar e instalar la aplicación directamente usando el archivo Android Package (APK).

## Pasos para Instalar

1.  **Descargar el APK**: Visitá la página de [Lanzamientos de KEDA](https://github.com/schoren/keda/releases) en tu dispositivo Android y descargá el archivo `keda_android_vX.Y.Z.apk` más reciente.
2.  **Abrir el Archivo**: Una vez que la descarga haya terminado, tocá el archivo en tu barra de notificaciones o en tu carpeta de "Descargas".
3.  **Permitir Apps Desconocidas**: Si se te solicita, permití que tu navegador o administrador de archivos pueda "Instalar aplicaciones desconocidas". Esta es una función de seguridad estándar de Android para aplicaciones que no se descargan del Play Store.
4.  **Confirmar Instalación**: Tocá **Instalar** y esperá a que el proceso termine.
5.  **Abrir KEDA**: Tocá **Abrir** o buscá el ícono de KEDA en tu menú de aplicaciones.


> [!NOTE]
> Es posible que necesites repetir este proceso para futuras actualizaciones, ya que la aplicación actualmente no admite actualizaciones automáticas cuando se instala vía APK.
